# syakyo-full-stack-fastapi-template

SYAKYO https://github.com/fastapi/full-stack-fastapi-template

## Tech Stack

- Frontend: React
  - Component: Chakra UI
  - Type: TypeScript
  - Build Tool: Vite
- Backend: FastAPI
  - Data Validation: Pydantic
  - ORM: SQLModel
  - Database: PostgreSQL
  - Unit Testing: pytest
- E2E Testing: Playwright
- Container: Docker
